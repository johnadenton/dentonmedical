Google Sheets Setup
ℹ️ Paste your Apps Script Web App URL below — both devices will sync automatically
How it works
• Google Sheets is your live database
• Every device pulls the latest data on open
• Every save pushes instantly to the cloud
• Receipt photos saved to Google Drive
• No more conflicts between devices ✓
Setup Steps (one time, on a computer)
1. Go to sheets.google.com → New blank spreadsheet
2. Name it: dentonmedical
3. Click Extensions → Apps Script
4. Delete all code, paste in google-apps-script.js
5. Click Deploy → New deployment → Web app
6. Execute as: Me · Access: Anyone
7. Click Deploy → Authorize → copy the URL
8. Paste URL below → tap Save & Connect
9. Tap Sync — all your data will appear!
